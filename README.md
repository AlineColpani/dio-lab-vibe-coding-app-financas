# Prompt Final

PRD – Aplicativo de Organização de Finanças Pessoais Conversacional

1. Contexto
Criar um aplicativo de organização de finanças pessoais que funcione por meio de conversas naturais com o usuário. O objetivo é simplificar o controle financeiro, eliminando a necessidade de formulários complexos ou planilhas manuais.

2. Problema
- Usuários desistem de controlar seus gastos porque os apps atuais exigem muita entrada manual.
- Falta de personalização e de uma experiência fluida.
- Necessidade de uma solução mais natural, acessível e adequada para iniciantes.

3. Público-Alvo
- Pessoas que desejam começar a organizar suas finanças de forma prática.
- Iniciantes que não têm familiaridade com planilhas ou apps tradicionais.
- Usuários que valorizam simplicidade, acessibilidade e interação amigável.

4. Princípios de Design
A solução deve seguir os princípios de Design Universal, garantindo acessibilidade e inclusão desde o início.
Aplicações práticas:
- Compatibilidade com leitores de tela.
- Linguagem clara e acessível.
- Contraste adequado e opções de personalização visual.
- Navegação simples para diferentes níveis de letramento digital.

5. Funcionalidades-Chave
1. Registrar gastos via chat em linguagem natural.
2. Classificação automática das transações.
3. Definição e acompanhamento de metas financeiras.
4. Possibilidade de incluir novas metas financeiras a qualquer momento.
5. Receber dicas de economia de um “Agente Financeiro” virtual.
6. Relatórios simples e personalizados, apresentados de forma visual e acessível.
7. Sistema de login para acesso seguro ao aplicativo.

6. Entregável da IA
- Plano de MVP contendo:
  - Principais telas (login, chat, metas, relatórios).
  - Recursos necessários (NLP, categorização automática, motor de recomendações, autenticação de usuários).
  - Esboço de validação inicial (testes com usuários iniciantes e com diferentes perfis de acessibilidade).
- Linguagem acessível e tom educativo em português.

# Conceito do App

Aplicativo de finanças pessoais de fácil interação através de chat que pode ser facilmente utilizado por pessoas de diferentes níveis de familiaridade com finanças.
